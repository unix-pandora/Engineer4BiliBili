# Engineer For BiliBili

Process audio and video files downloaded from Bilibili Windows PC end.

<hr>

# Docs

- 参数预设定: `paramters`

- 事先要先创建: **output_folder**

- 执行 `summarize`,会生成 `summarizes.json`。修订`type`,默片:`silence`,有声:`voice`

- 执行分流器: `splitter`

- 若现编码故障: inspect `paramters.unicode_chars`
